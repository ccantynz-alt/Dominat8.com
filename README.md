# Dominat8.com – The Marketing Monster

The AI Website Builder for the 1%. A futuristic landing page built with Next.js 15, designed for year 2030 aesthetics.

## Tech Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **React Three Fiber** – 3D AuraSphere with glassmorphism
- **Framer Motion** – Animations and micro-interactions
- **Lenis** – Smooth scrolling

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

- `src/app/` – App Router pages and layout
- `src/components/` – Hero, AuraSphere, Features, ComparisonSlider, Footer, CustomCursor, SmoothScroll
- `src/assets/` – Static assets

## Features

- **AuraSphere** – Floating 3D sphere with purple/blue glow and mouse parallax
- **Hero** – Gradient headline with fog-reveal animation
- **Features** – Glass-style cards with border beam effect
- **Comparison Slider** – Before vs. Dominat8 interactive slider
- **Custom Cursor** – Glowing ring that expands on hover
- **Lenis** – Smooth scroll
- **Footer** – Bold outlined “DOMINAT8” typography
